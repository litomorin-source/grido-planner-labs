# GridoPlanner Labs

Laboratorio de desarrollo para GridoPlanner.

## Versión

Labs 0.1-alpha

## Objetivo

Verificar conexión con GitHub desde Streamlit usando Secrets.

## Secrets necesarios en Streamlit

```toml
GITHUB_TOKEN = "tu_token"
GITHUB_REPO = "litomorin-source/grido-planner-labs"
GITHUB_BRANCH = "main"
```
